# NonogramOnline

### How to Play
- Open the **sbt** shell
- Run the **run** or **~fastLinkJS** command
- Just open the **NonogrammOnline.html** file in your Browser

### Features
- Play the game
- Solver

### Modes 
- Five Lives
- Hardcore

### Sizes
- 5x5
- 10x10
- 15x15
- 20x20

### Suported Browsers
- Chrome

### Versions
- sbt: 1.4.5
- scalaVersion: 2.13.4
